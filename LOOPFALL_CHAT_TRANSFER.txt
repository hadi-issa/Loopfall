PROJECT OVERVIEW
============================================================

Project name: Loopfall
Engine: Unity 3D
Genre: narrative-driven atmospheric exploration prototype
Core themes: time loops, decay, memory, emotional processing
Combat: none

Player concept:
- The player is a physical ball, not a humanoid.
- Movement must be Rigidbody/physics based.
- Movement should feel responsive but slightly unstable.
